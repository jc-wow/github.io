javascript中asnyc的一点理解

发表于 2019-09-11 | 更新于: 2019-10-15

工作中遇见了一个用到asnyc的小问题，需求是要按次序请求两个Ajax接口。由于是第一次使用asnyc，顺便把解决过程中遇到的问题纪录下来。

1.当执行asnyc函数的时候，一旦遇到await就会先返回，等到异步操作完成，再执行函数体后面的语句。因此执行getAllData函数的时候遇到await先执行完ajaxRequest1函数，然后再执行ajaxRequest2函数，这样就实现了异步。
2.await命令后通常是Promise对象，返回该对象的结果。因此ajaxRequest1()的返回值应该为Promise对象。
3.Promise函数接受一个函数作为参数，该函数的两个参数分别是resolve和reject，它们是两个函数，由JavaScript引擎提供，不用自己部署，直接调用。因此，只有调用了resolve函数，Promise函数的resolved状态才会传给getAllData函数await之后的Promise对象的返回值中。
